Friend Function in C++

First, let's understand what a friend function is?

1. A friend function is not a member function which means ideally it should not have any right to access private members of a class but it can.

2. A friend function is defined outside class and to declare it keyword friend is used.

3. Friend function is able to access private and protected members of a class.

Let's understand this with an example:

#include<iostream>

using namespace std;

class sum{

private:

  int side1,side2;

public:

  void setwidth(int a,int b){

   side1 = a;

   side2 = b;

  }

friend void showwidth(sum x); // using friend keyword

};

void showwidth(sum o1)

{

  cout<<"The sum is "<<o1.side1+o1.side2;

}

int main()

{

sum o1;

o1.setwidth(2,3);

showwidth(o1);

return 0;

}

Output

The sum is 5
