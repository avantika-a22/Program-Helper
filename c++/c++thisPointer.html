This pointer .

Before jumping to program, let's first understand how objects see functions and datamembers.

Objects can be numerous of a particular class that have properties of the class.

1. Each object creates its own copy of the data member.

2. Same functions are accessed by all the objects.

Now if all the objects share same function definition, how data is managed and updated differently in different objects?

This is done with the help of this pointer.

This pointer is a pointer which contains address of the object and is passed as a hidden argument to all non-static member functions.

It is available as local variable within the body of all nonstatic functions.

#include<iostream>

using namespace std;

class sample

{

  private:

    int h,r;

  public:

    void setdata(int m,int n)

    {

      this->h = m;

      this->r = n;

    }

    void display()

    {

      cout<<"h==" << h << endl;

      cout<<"r==" << r << endl;

    }

};

int main()

{

  sample s1,s2;

  s1.setdata(10,20);

  s2.setdata(20,40);

  s1.display();

  s2.display();

  return 0;

}

Output

h==10

r==20

h==20

r==40
