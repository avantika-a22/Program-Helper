C++ Program for Banking(ATM) System.

To code a banking or ATM system which includes functions like -

1. Deposit money

2. Withdraw money

3. View balance

4. Change pin

1. First part is to add a deposit money function -

Let's have a look at how to create this function -

First we need to declare global variables for pin, balance and amount.Also we will use our file functions so that we can store our data(balance , pin) in files.

int balance;

int amount;

int pin;

These above three variables are global variables.

Now, let us see the main function and then deposit money function -

In the above code, we have created(if not it automatically gets) a file named data.dat which stores net amount and other file pin.dat which stores pin of the user.

As the code runs, it first asks for pin ,if pin entered matches with the pin in file(pin.dat) it gives you access to other functions.

Now let's see deposit money function -

In the above code, we can clearly see that if we want to add money it automatically adds it to the balance variable(balance variable's value is stored in file data.dat).

2. Second function is withdraw function

The function is similar to deposit except amount is deducted from balance here.

3. Third is view balance function -

View balance is a simple ones just to print value of balance variable.

4. Fourth is change pin function -

It includes reading data from file pin.dat saved in the same folder and writing new pin to it.

K in the above code is also a globally declared variable.

Now let's see the entire code -

#include<iostream>

#include<fstream>

int balance = 0;

int amount ;

int pin;

int k;

using namespace std;

int withdraw()

  {

    if(balance<100)

      {

        cout<<"insufficient balance";

        balance = balance;

      }

    else

      {

      cout<<"enter the amount to withdraw: "<<endl;

      cin>>amount;

      if(amount%100!=0)

          {

          cout<<"\n"<<"please enter in multiple of 100";

          balance = balance;

          }

      else if(balance<amount)

          {

          cout<<"Insufficient balance"<<endl;

          balance = balance;

          }

      else

        {

        balance -=amount;

        return(balance);

        }

      }

  }

void view()

  {

    cout<<balance<<endl;

  }

int deposit()

  {

    cout<<"Enter the amount you want to deposit: ";

    cin>>amount;

    balance +=amount;

    return(balance);

  }

void change()

  {

    int p;

    cout<<"Please enter your old password: ";

    cin>>p;

    if (p==pin)

    {

      cout<<"Enter new password: "<<endl;

      cin>>k;

      pin=k;

    }

    else{

      cout<<"wrong password";

    }

  ofstream fout;

  fout.open("pin.dat");

  fout<<pin;

  }

int main()

{

int choice,password;

cout<<"*************WELCOME*************";

cout<<"Enter your password: "<<endl;

cin>>password;

ifstream fin;

fin.open("data.dat");

fin>>balance;

ifstream f;

f.open("pin.dat");

f>>pin;

cout<<pin;

if(password!=pin)

{

cout<<"wrong password!"<<endl;

}

else{

cout<<"1-withdraw money,2-view balance,3-change password,4-deposit,5-exit"<<endl;

while(choice!=5)

{

cout<<"enter your choice: "<<endl;

cin>>choice;

switch(choice)

{

  case 1:

    balance=withdraw();

    break;

  case 2:

    view();

    break;

  case 3:

    change();

    break;

  case 4:

    balance=deposit();

    break;

  default:

  cout<<"invalid option";

  ofstream fout;

  fout.open("");

  fout<<balance;

}

}

}

return 0;

}
