Structure and Union

Structure

Let's try to first understand what structure is ?

For e.g. we have to maintain records of students with each student having three different attributes which can be name, age, class.In this , name is string, age and class are integers. Now all the elements of students record are not similar(their type is not similar) , here arrays cannot be used because we know, arrays are collection of same type of data.

So now to store these record of students, we'll use structure, which is capable of storing different types of data. Data of different types can be grouped together under a single name using structure

struct name{

  datatype member1;

  datatype member2;

  datatype member3;

  datatype member4;

};

Here struct is a keyword, which tells the compiler that a structure is being defined.

And members are name,class,salary etc;

For e.g:-

    struct student{

          char name[15];

          int age;

          float marks;

  }stu1, stu2, stu3;

now stu1 ,stu2, stu3 are of type struct.

 Union

Union is similar to structure and it can also contain members of different data types. The syntax used for definition of a union, declaration of union variables and for accessing members is similar to structure.

 Keyword union is used in place of struct in union.

The main difference between union and structure is in the way memory is allocated for the members. In a structure each member has its own memory location, whereas members of union share the same memory location. When a variable of type union is declared, compiler allocates sufficient memory to hold the largest member in union. Since all members share the same memory location hence we can use only one member at a time. Thus union is used for saving memory.

The syntax of definition of a union is:-

struct union name{

    datatype member 1;

    dataype member 2;

     ........

};

union result{

   int marks;

   char grade;

   float per;

   }variable name;
