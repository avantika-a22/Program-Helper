File Handling in C

Generally when we terminate our programs, all the data is lost because its in volatile memory.If the data has to be used later, then it becomes necessary to keep it in permanent storage device. C support the concept of files through which data can be stored on the disk or secondary storage device. The stored data can be read whenever required.

There are two ways of storing data on files first is text mode and second is binary mode. Text mode is easily readably by humans but binary mode is not readable.

The three basic steps for file operations are :

- Opening of file

- Read or write

- Closing the file

1. To open a file fopen() function is used.For example:- fopen("myfile.txt","w");

2. To close a file fclose() function is used.

Now there are multiple modes in which files can be opened:-

1. "w" (write) - If the file doesn't exist then this mode creates a new file for writing, and if the file already exists then the previous data is erased and the new data entered is written to the file.

2. "a" (append) - If the file doesn't exist then this mode creates a new file, and if the file already exists then the new data entered is appended at the end of existing data. In this mode, the data existing in the file is erased as in "w" mode.

3. "r" (read) - It is used for opening an existing file for reading purpose only. The file to be opened must exist and the previous data of file is not erased.

4. "w+" (write + read) - It is same as "w" mode but in this mode we can also read and modify the data. If the file doesn't exist then a new file is created and if the file exists then previous data is erased.

5. "r+" (read + write) - This mode is same as "r" mode but in this mode we can also write and modify existing data. The file that has to be opened must exist and the previous data of file is not erased. Since we can add new data modify existing data so this mode is also called update mode

6. "a+" (append + read) - It is same as the "a" mode but in this we can also read the data stored in the file. If the file doesn't exist, a new file is created and if the file already exists then new data is appended at the end of existing data. We cannot modify existing data in this mode.

 Program to understand File functions.

#include<stdio.h>

#include<conio.h>

int main()

{

  FILE *fp ;

  char data[50] ;

  printf( "Opening the file myFile.c in read mode") ;

  fp = fopen( "myFile.c", "r" ) ;

  if ( fp == NULL )

    {

      printf( "Could not open file" ) ;

      return 1;

    }

  printf( "Reading the file myFile.c" ) ;

  while( fgets ( data, 50, fp ) != NULL )

  printf( "%s" , data ) ;

  printf("Closing the file") ;

  fclose(fp) ;

  return 0;

}

Other functions in File Handling

getw() - It reads an integer from a file

putw() - It writes an integer to file

fgetc() - It reads a character from file.

fputc() - It write a character to file.

fgets() - It reads string from a file, one line at a time.

fputs() - It writes string to a file.

gets() - It reads line from keyboard.

puts() - It writes line to output screen.

sprintf() - It writes formatted output to string.

remove() - This function deletes a file.

fflush() - This function flushes a file.

ftell() - It returns current position of file pointer

fseek() - This function moves file pointer position to given location.

fscanf() - It reads formatted data from a file.

fprintf() - This function writes formatted data to a file.

rewind() - This function moves file pointer position to the beginning of the file.

getc() - It reads character from file.

SEEK_SET - This function moves file pointer position to the beginning of the file.

SEEK_CUR - This function moves file pointer position to given location.

SEEK_END - This function moves file pointer position to the end of file.

sscanf() - It reads formatted input from a string.

fgetchar() - This function reads a character from keyboard.
