Functions

- A functions is a block of statements that is used to carry out a specific task.It is actually a subprogram.

Now let's try to understand why to use functions and what are advantages of using it?

1. If we talk about programming , there are lines of codes which can be huge in numbers.So program is basically divided and it is achieved with use of functions.

2. The main reason is that if a piece of code is to be used a number of times in different places in program, we use functions.In this way repetition of codes is avoided.

3. The program becomes easy to understand, modifiable and easy to debug and test.

Now let's discuss about types of functions in C:-

1. Library functions

2. User-defined functions

  Library functions

In C, there are some predefined functions which are already there in C library.For example printf(), scanf(), strcmp() etc. are some functions that are alrady predefined and we can use them in our program.

To use a library function, we have to include corresponding header file(stdio.h,conio.h,math.h etc.) using the preprocessor directive #include.

C Program to find square root of a number

#include<stdio.h>

#include<conio.h>

#include<math.h>

void main()

  {

     double n,root;

     printf("Enter the number: ");

     scanf("%lf",&n);

     root = sqrt(n);

     printf("Square root of number is %.2lf",root);    // upto two decimal places

     getch();

}

Output

Enter a number: 25

Square root of number is 5.00

{

Function Arguments

The calling function sends some values to the called function for communication; these values are called arguments or parameters

Actual arguments

: The arguments which are mentioned in the function cali are known as actual arguments, since these are the values which are actually sent to the called function. Actual.arguments can be written in the form of variables, constants or expressions or any function call that returns a value, For example:- In the above program, number is the actual argument

Formal arguments

The name of the arguments, which are mentioned in the function definition are called formal or dummy arguments since they are used just to hold the values that are sent by the calling function.

For example,in the above program, num is the formal argument
