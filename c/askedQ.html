Q. Write the output of the following statements

     int a = 12,b = 20;

     System.out.println("a++ = " +a++);

     System.out.println("--a = " +--a);

Solution :-   a++ = 12

                  --a = 12

After a++ the value of a will be incremented by 1 which makes it 13,but output will be 12 as it is post-increment and now the value of a is 13,then with --a first value is decreased by 1 which makes it 12 and output will be 12 again.
