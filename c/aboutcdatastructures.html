Data Structures

Data Structures as the name tells, are used to structure and store data in an organised and efficient manner. The C Programming language has many data structures like an array, stack, queue, linked list, tree, etc. A programmer selects an appropriate data structure and uses it according to their convenience.

Arrays, queue, linked list we have already discussed in previous sections.Now let's try to understand trees and hashing.

 Trees

A tree is a data structure that has one root node and many sub-nodes. It is another one of the data structures which are designed on top of a linked list.

 Hashing

It is used to implement an associative array, a structure that can map keys to values. Hash table uses a hash function to compute an index into an array of buckets. Hash tables are very useful data structures.
