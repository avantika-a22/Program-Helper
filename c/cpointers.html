Pointers

- The concept of pointers is very important if you really want to understand things, behind the scenes how actually memory management takes place , abour variables and many more things.

Let's first try to understand how memory is organized in a computer.The memory is made up of bytes arranged in a sequential manner.

Jump directly to example programs related to pointers.

Each byte has an index number, which is called the address of that byte. The address of these bytes start from zero and-the address of last byte is one less than the size of memory

We have studied that it is necessary to declare a variable before using it, since compiler has to reserve space for it. The data type of the variable also has to be mentioried so that the compiler knows how much space needs to be reserved

    Let's try to understand this with declaration of variable.

Let us suppose we declare a variable age of type int : -

      int age;

The compiler reserves 2 consecutive bytes from memory for this variable and associates the name age with it. The address of first byte from the two allocated bytes is known as the address of variable age.

If you have stored a value of type integer in a variable like :-

    int age = 60;

Now the value 20 will be stored in two consecutive bytes say for ex.(1222 and 1223) in binary form.Similarly for char, 1 byte is allocated, float is allocated 4 bytes and so on.Now the address of first byte, in this case, 1222 is the address of variable.

Pointer Variables

Pointer variables need to be declared before they can be used.The declaration is same as of any other variable,

     int *pointername;

     char *pointername;

     float *p1;

One important thing that needs to be understood is, that pointers are also variables so compiler will reserve space for them and pointer variable will also have some address.

{

(*x)++;

(*y)++;

}

int main()

{

  int a = 5,b = 6;
