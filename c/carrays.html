Arrays

- Variables we use can store only one variable at a time,but there are situations when we want to store more than one value.

For example, we want to store roll no. of 50 students in a class.Normally to perform this task we would :-

1. Declare 50 variables to store roll no.

2. Assigning value to each variable.

3. Display the value of each variable.

   We can carry out our task with these 3 steps but this is not efficient way of doing it and it would be difficult to assign value one by one in cases where we have to work with large data.

Consider a situation where information of 1000 employees is to be recorded.Here 1000 variables would be used in case of not using array, but array makes it simple.

How? Let's see!

Basically, an array is a collection of similar type of data items and each data item is called an element of the array.

The element of array share the same variable name but each element has a different index number known as subscript.

Like we discussed about storing roll no of 50 students, we can take an array variable rollNo[50] of type int.The size of this array variable is 50, so it can store upto 50 integer values.

The individual elements of this array are: -

    rollNo[0], rollNo[1], rollNo[2], rollNo[3]................rollNo[50]

In C,the subscripts start from zero, so rollNo[O] is the first element, rollNo[ 1] is the second element of array and so on.
